Acuity Scheduling Account Setup

How to create and connect your Acuity account with Zoho CRM

1

Create Acuity Scheduling Account

Visit acuityscheduling.com and sign up for a new account. Choose the plan that fits your business needs - Basic (Free), Emerging ($14/month), Growing ($23/month), or Powerhouse ($45/month).

2

Install Zoho CRM Extension

Go to Zoho Marketplace and install the "Acuity Scheduling for Zoho CRM" extension by Ulgebra. This integration allows seamless appointment management within your CRM.

3

Configure API Authentication

In Acuity, go to Business Settings → Integrations → API to get your User ID and API Key. Enter these credentials in the Zoho CRM extension settings to establish the connection.

security Authentication Details

API Authentication:

Acuity uses HTTP Basic Auth with your numeric User ID as username and API Key as password. The API endpoint is: https://acuityscheduling.com/api/v1/

Rate Limits:

• 10 requests per second
• 20 concurrent connections
• Maximum 25 webhooks per account

Security:

All API requests must be made via HTTPS. Keep your API credentials secure and never expose them in client-side code.

Frequently Asked Questions

Find answers to common questions about your viewing-only dashboard

This dashboard is designed for viewing and analyzing your Acuity Scheduling data within Zoho CRM. You can view appointments, analyze booking trends, search through data, export reports, and monitor your business performance. However, you cannot create, edit, or delete appointments directly from this interface.

Click on "Analytics Dashboard" in the sidebar to view comprehensive metrics including revenue trends, appointment counts, service performance, and client analytics. The dashboard displays interactive charts and key performance indicators to help you understand your business patterns.

Use the search bar in the top navigation to search by client name, appointment type, or date. Each section (Upcoming, Past, Services) also has advanced filtering options including date ranges, service types, and booking status. You can combine multiple filters for precise results.

Yes! You can export data from multiple places: Use the export button on the Analytics Dashboard for comprehensive reports, or visit individual sections (Past Appointments, Services) and use their respective export functions. Data is exported in CSV format with detailed information including client details, booking history, and revenue data.

Use the theme switcher button (palette icon) in the bottom right corner. You can switch between light and dark modes, and choose from 8 different color themes including blue, purple, green, orange, red, teal, pink, and indigo to personalize your viewing experience.

This dashboard is designed as a read-only analytics and monitoring interface. To create, edit, or manage appointments, you need to use your main Acuity Scheduling account at acuityscheduling.com or through direct Acuity integrations. This separation ensures data integrity while providing powerful viewing and analysis capabilities.

The dashboard data is synchronized with your Acuity Scheduling account in real-time through webhooks and API calls. New appointments, cancellations, and updates typically appear within minutes. You can also manually refresh any section using the refresh button in the top navigation.
