You can choose the module into which the new Acuity clients data should be added in Zoho CRM by selecting Contacts OR Leads in the drop-down near 'Add new Calendly Clients to'. First Name, Last Name, E-mail address and Phone number will be added to the record from the data received from Acuity Scheduling.

Click on the drop-down near 'Add Appointments to' and select Events OR Tasks into which Acuity's appointments should be added in Zoho CRM. Events will be created with the appointment start time and end time in the 'From' and 'To' fields respectively. For Tasks, only start time will be added in the 'Due Date' field.

Enable OR Disable sales signal notifications () for new appointments by clicking on the switch () next to 'New Appointment Notifications'.
